Credit Card Fraud Detection

This project builds a machine learning model to detect fraudulent credit card transactions using a real-world transaction dataset. The data is preprocessed by scaling numerical features, analyzing class distribution, and handling class imbalance through oversampling to improve fraud detection.

The project compares Logistic Regression and Random Forest classifiers to identify fraudulent and genuine transactions. Model performance is evaluated using accuracy, precision, recall, F1-score, confusion matrix, and classification report. The Random Forest model achieved the best overall performance with high accuracy and effective fraud detection.

This project demonstrates how Python libraries such as pandas, NumPy, matplotlib, seaborn, scikit-learn, and imbalanced-learn can be used together to solve a real-world fraud detection problem.

Key Features

* Data loading and preprocessing
* Feature scaling and normalization
* Handling class imbalance using oversampling
* Train-test data splitting
* Logistic Regression and Random Forest models
* Model evaluation using confusion matrix and classification report

Technologies Used

* Python
* pandas
* NumPy
* matplotlib
* seaborn
* scikit-learn
* imbalanced-learn
